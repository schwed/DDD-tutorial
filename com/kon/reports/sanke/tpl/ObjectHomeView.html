Metadata Explorer Technical metadata management and navigation tool

Object tree shows list of all root objects in MMX repository which has no defined parent. Root objects are grouped by OBJECT TYPE and the number in the end of object name indicates the count of child objects: Object Name (child count)
Object visibility (public_ind) is coded with (public:1) or (private:0).
Object state (state_ind) is coded with (active:1) or (deleted:0).
